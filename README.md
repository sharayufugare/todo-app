This is a full-stack Todo application built using Spring Boot, Spring Security, and PostgreSQL, with a simple HTML/CSS/JavaScript frontend.
The application allows users to register, login, and manage their personal todo list with full CRUD operations (Create, Read, Update, Delete).

User authentication is handled securely using session-based login, and data is stored in a PostgreSQL database deployed on Render.
The project is fully deployed on Render, demonstrating real-world backend deployment, database integration, and environment-based configuration.
